# NyanBot
This is a [will](https://github.com/skoczen/will) bot.

Requirments -
Will -> 
```
pip install will
```

Redis -> 
```
apt-get install redis-server
```

To launch, run ```run_will.py```, it should walk you through what environment variables need to be set up to connect to Hipchat.
